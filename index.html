Taller Guiado: Solución de Sistemas de Ecuaciones por Eliminación

Introducción al Método de Eliminación

Este taller guiado presenta la solución paso a paso de diversos sistemas de ecuaciones utilizando exclusivamente la técnica de eliminación (también conocida como método de Gauss-Jordan). Este método consiste en transformar el sistema original en un sistema equivalente más simple mediante operaciones elementales:

  • Multiplicar una ecuación por un escalar no nulo
  • Sumar o restar múltiplos de ecuaciones para eliminar variables
  • Intercambiar el orden de las ecuaciones

El objetivo es obtener un sistema escalonado donde podamos identificar fácilmente la solución o determinar si el sistema tiene infinitas soluciones o ninguna solución.

Ejercicio 1: Sistema Lineal 2×2 con Solución Única ▼
Sistema original:
\begin{align} x - 4y + 1 &= 0 \quad (1)\\ 3x + 2y - 1 &= 0 \quad (2) \end{align}
Concepto: Reacomodar las ecuaciones para tener los términos con variables a la izquierda

Reorganizamos las ecuaciones para seguir el formato estándar con los términos constantes al lado derecho.

\begin{align} x - 4y &= -1 \quad (1)\\ 3x + 2y &= 1 \quad (2) \end{align}
Concepto: Eliminar la variable x de la segunda ecuación

Multiplicamos la ecuación (1) por 3 para que el coeficiente de x coincida con el de la ecuación (2), y luego restamos.

\begin{align} 3(x - 4y) &= 3(-1)\\ 3x - 12y &= -3 \quad (1') \end{align}

Ahora restamos (1') de (2) para eliminar x:

\begin{align} (3x + 2y) - (3x - 12y) &= 1 - (-3)\\ 3x + 2y - 3x + 12y &= 1 + 3\\ 14y &= 4\\ y &= \frac{4}{14} = \frac{2}{7} \end{align}
Concepto: Sustitución hacia atrás

Una vez encontrado el valor de y, lo sustituimos en la ecuación (1) para encontrar el valor de x.

\begin{align} x - 4y &= -1\\ x - 4\left(\frac{2}{7}\right) &= -1\\ x - \frac{8}{7} &= -1\\ x &= -1 + \frac{8}{7}\\ x &= \frac{-7 + 8}{7} = \frac{1}{7} \end{align}
Solución:

El sistema tiene una única solución:

\begin{align} (x, y) = \left(\frac{1}{7}, \frac{2}{7}\right) \end{align}

Ejercicio 2: Sistema Lineal 2×2 Sin Solución ▼
Sistema original:
\begin{align} x - 2y &= 6 \quad (1)\\ -0.5x + y &= 1 \quad (2) \end{align}
Concepto: Eliminar fracciones en la segunda ecuación

Multiplicamos la ecuación (2) por 2 para eliminar la fracción.

\begin{align} x - 2y &= 6 \quad (1)\\ 2(-0.5x + y) &= 2(1)\\ -x + 2y &= 2 \quad (2') \end{align}
Concepto: Eliminar la variable x

Sumamos las ecuaciones (1) y (2') para eliminar la variable x.

\begin{align} (x - 2y) + (-x + 2y) &= 6 + 2\\ 0 &= 8 \end{align}

Obtenemos una contradicción (0 = 8), lo que significa que el sistema no tiene solución.

Concepto: Interpretación geométrica

Geométricamente, este sistema representa dos líneas paralelas en el plano que nunca se intersectan, por lo tanto, no hay puntos en común que satisfagan ambas ecuaciones simultáneamente.

Solución:

El sistema no tiene solución.

Ejercicio 3: Sistema Lineal 2×2 con Solución Única ▼
Sistema original:
\begin{align} x - y &= 2 \quad (1)\\ x + y &= 1 \quad (2) \end{align}
Concepto: Eliminar la variable x mediante suma de ecuaciones

Restamos la ecuación (2) de la ecuación (1) para eliminar la variable x.

\begin{align} (x - y) - (x + y) &= 2 - 1\\ x - y - x - y &= 1\\ -2y &= 1\\ y &= -\frac{1}{2} \end{align}
Concepto: Sustituir el valor encontrado

Sustituimos el valor de y en la ecuación (2) para encontrar x.

\begin{align} x + y &= 1\\ x + \left(-\frac{1}{2}\right) &= 1\\ x &= 1 + \frac{1}{2}\\ x &= \frac{3}{2} \end{align}
Concepto: Verificación de la solución

Comprobamos que los valores obtenidos satisfacen ambas ecuaciones originales.

Ecuación (1): \(x - y = 2\)

\begin{align} \frac{3}{2} - \left(-\frac{1}{2}\right) &= \frac{3}{2} + \frac{1}{2} = 2 \checkmark \end{align}

Ecuación (2): \(x + y = 1\)

\begin{align} \frac{3}{2} + \left(-\frac{1}{2}\right) &= \frac{3}{2} - \frac{1}{2} = 1 \checkmark \end{align}
Solución:

El sistema tiene una única solución:

\begin{align} (x, y) = \left(\frac{3}{2}, -\frac{1}{2}\right) \end{align}

Ejercicio 4: Sistema Lineal 2×2 con Infinitas Soluciones ▼
Sistema original:
\begin{align} -x - 2y &= -4 \quad (1)\\ 5x + 10y &= 20 \quad (2) \end{align}
Concepto: Simplificar la segunda ecuación

Dividimos la ecuación (2) por 5 para simplificar los coeficientes.

\begin{align} -x - 2y &= -4 \quad (1)\\ \frac{5x + 10y}{5} &= \frac{20}{5}\\ x + 2y &= 4 \quad (2') \end{align}
Concepto: Sumar ecuaciones para eliminar variables

Sumamos las ecuaciones (1) y (2') para eliminar variables.

\begin{align} (-x - 2y) + (x + 2y) &= -4 + 4\\ 0 &= 0 \end{align}

Obtenemos una identidad (0 = 0), lo que significa que las dos ecuaciones son linealmente dependientes y representan la misma recta. Por tanto, el sistema tiene infinitas soluciones.

Concepto: Expresar la solución en forma paramétrica

Podemos expresar la solución en términos de un parámetro. Despejamos x de la ecuación (2').

\begin{align} x + 2y &= 4\\ x &= 4 - 2y \end{align}
Solución:

El sistema tiene infinitas soluciones, que pueden expresarse como:

\begin{align} \{(x, y) \mid x = 4 - 2y, y \in \mathbb{R}\} \end{align}

O alternativamente:

\begin{align} \{(x, y) \mid -x - 2y = -4\} \end{align}

Geométricamente, todas las soluciones están sobre la recta \(-x - 2y = -4\).

Ejercicio 5: Sistema Lineal 3×3 con Solución Única ▼
Sistema original:
\begin{align} x + y - z &= 0 \quad (1)\\ x - y + z &= 2 \quad (2)\\ 2x + y - 4z &= -8 \quad (3) \end{align}
Concepto: Eliminar la variable x de las ecuaciones (2) y (3)

Restamos la ecuación (1) de la ecuación (2) para eliminar x.

\begin{align} (x - y + z) - (x + y - z) &= 2 - 0\\ -2y + 2z &= 2\\ -y + z &= 1 \quad (4) \end{align}
Concepto: Continuar con la eliminación

Restamos 2 veces la ecuación (1) de la ecuación (3) para eliminar x.

\begin{align} (2x + y - 4z) - 2(x + y - z) &= -8 - 0\\ 2x + y - 4z - 2x - 2y + 2z &= -8\\ -y - 2z &= -8\\ -y - 2z &= -8 \quad (5) \end{align}
Concepto: Resolver el sistema de ecuaciones resultante

Ahora tenemos un sistema de ecuaciones en y y z con las ecuaciones (4) y (5).

Despejamos y de la ecuación (4):

\begin{align} -y + z &= 1\\ y &= z - 1 \quad (6) \end{align}

Sustituimos esta expresión en la ecuación (5):

\begin{align} -(z - 1) - 2z &= -8\\ -z + 1 - 2z &= -8\\ -3z + 1 &= -8\\ -3z &= -9\\ z &= 3 \end{align}
Concepto: Sustituir hacia atrás

Con el valor de z, encontramos y utilizando la ecuación (6):

\begin{align} y &= z - 1\\ y &= 3 - 1 = 2 \end{align}

Con los valores de y y z, encontramos x utilizando la ecuación (1):

\begin{align} x + y - z &= 0\\ x + 2 - 3 &= 0\\ x &= 1 \end{align}
Concepto: Verificación de la solución

Comprobamos que los valores obtenidos satisfacen las ecuaciones originales.

Ecuación (1): \(x + y - z = 0\)

\begin{align} 1 + 2 - 3 = 0 \checkmark \end{align}

Ecuación (2): \(x - y + z = 2\)

\begin{align} 1 - 2 + 3 = 2 \checkmark \end{align}

Ecuación (3): \(2x + y - 4z = -8\)

\begin{align} 2(1) + 2 - 4(3) = 2 + 2 - 12 = -8 \checkmark \end{align}
Solución:

El sistema tiene una única solución:

\begin{align} (x, y, z) = (1, 2, 3) \end{align}

Ejercicio 8: Sistema Lineal 3×3 Sin Solución ▼
Sistema original:
\begin{align} x + 7y - 4z &= 1 \quad (1)\\ 2x + 3y + z &= -3 \quad (2)\\ -x - 18y + 13z &= 2 \quad (3) \end{align}
Concepto: Eliminar la variable x

Sumamos la ecuación (1) multiplicada por 2 a la ecuación (2) para eliminar x.

\begin{align} 2(x + 7y - 4z) + (2x + 3y + z) &= 2(1) + (-3)\\ 2x + 14y - 8z + 2x + 3y + z &= 2 - 3\\ 4x + 17y - 7z &= -1 \quad (4) \end{align}
Concepto: Continuar eliminando variables

Sumamos la ecuación (1) a la ecuación (3) para eliminar x.

\begin{align} (x + 7y - 4z) + (-x - 18y + 13z) &= 1 + 2\\ -11y + 9z &= 3 \quad (5) \end{align}
Concepto: Despejar variables en ecuaciones reducidas

Despejamos -11y de la ecuación (5).

\begin{align} -11y + 9z &= 3\\ -11y &= 3 - 9z \quad (6) \end{align}
Concepto: Usar el método de eliminación para verificar si el sistema es consistente

Multiplicamos la ecuación (1) por 4 y la sumamos a la ecuación (4).

\begin{align} 4(x + 7y - 4z) + (4x + 17y - 7z) &= 4(1) + (-1)\\ 4x + 28y - 16z + 4x + 17y - 7z &= 4 - 1\\ 8x + 45y - 23z &= 3 \quad (7) \end{align}

Ahora multiplicamos la ecuación (5) por -1.

\begin{align} -1 \cdot (-11y + 9z) &= -1 \cdot 3\\ 11y - 9z &= -3 \quad (8) \end{align}

Según las ecuaciones (5) y (8), tenemos:

\begin{align} -11y + 9z &= 3 \quad (5)\\ 11y - 9z &= -3 \quad (8) \end{align}

Si sumamos estas ecuaciones, obtenemos:

\begin{align} (-11y + 9z) + (11y - 9z) &= 3 + (-3)\\ 0 &= 0 \end{align}

Esto nos da una identidad (0 = 0), lo que sugiere que las ecuaciones son linealmente dependientes. Necesitamos verificar con la ecuación (7).

Concepto: Verificar la consistencia del sistema completo

Usemos las ecuaciones (7) y (5) para verificar si hay solución.

Expresemos la ecuación (7) en términos de y y z.

De la ecuación (1), podemos despejar x:

\begin{align} x &= 1 - 7y + 4z \end{align}

Sustituyendo en la ecuación (7):

\begin{align} 8(1 - 7y + 4z) + 45y - 23z &= 3\\ 8 - 56y + 32z + 45y - 23z &= 3\\ 8 - 11y + 9z &= 3\\ -11y + 9z &= 3 - 8\\ -11y + 9z &= -5 \end{align}

Pero según la ecuación (5), \(-11y + 9z = 3\). Así que llegamos a una contradicción:

\begin{align} -11y + 9z &= -5\\ -11y + 9z &= 3 \end{align}

Esta contradicción (\(-5 \neq 3\)) indica que el sistema no tiene solución.

Solución:

El sistema no tiene solución.

Geométricamente, esto significa que los tres planos representados por las ecuaciones originales no se intersectan en un punto común ni en una línea común.

Ejercicio 10: Sistema con Fracciones ▼
Sistema original:
\begin{align} -x + 3y + 2z &= 2 \quad (1)\\ \frac{1}{2}x - \frac{3}{2}y - z &= -1 \quad (2)\\ -\frac{1}{3}x + y + \frac{2}{3}z &= \frac{2}{3} \quad (3) \end{align}
Concepto: Eliminar fracciones multiplicando por el mínimo común múltiplo

Multiplicamos la ecuación (2) por 2 para eliminar el denominador 2.

\begin{align} 2\left(\frac{1}{2}x - \frac{3}{2}y - z\right) &= 2(-1)\\ x - 3y - 2z &= -2 \quad (2') \end{align}
Concepto: Continuar eliminando fracciones

Multiplicamos la ecuación (3) por 3 para eliminar el denominador 3.

\begin{align} 3\left(-\frac{1}{3}x + y + \frac{2}{3}z\right) &= 3\left(\frac{2}{3}\right)\\ -x + 3y + 2z &= 2 \quad (3') \end{align}
Concepto: Analizar el sistema resultante

Observamos que las ecuaciones (1), (2') y (3') son:

\begin{align} -x + 3y + 2z &= 2 \quad (1)\\ x - 3y - 2z &= -2 \quad (2')\\ -x + 3y + 2z &= 2 \quad (3') \end{align}

Notamos que la ecuación (1) y la ecuación (3') son idénticas. Además, la ecuación (2') es igual a la ecuación (1) multiplicada por -1. Esto significa que las tres ecuaciones son linealmente dependientes y representan el mismo plano en el espacio 3D.

Concepto: Determinar el tipo de solución

Cuando las ecuaciones de un sistema son linealmente dependientes y representan el mismo plano, el sistema tiene infinitas soluciones. Estas soluciones pueden expresarse de forma paramétrica.

Podemos expresar una variable en términos de las otras dos. Por ejemplo, despejando x de la ecuación (1):

\begin{align} -x + 3y + 2z &= 2\\ -x &= 2 - 3y - 2z\\ x &= -2 + 3y + 2z \end{align}
Solución:

El sistema tiene infinitas soluciones, que pueden expresarse como:

\begin{align} \{(x, y, z) \mid x = -2 + 3y + 2z, y \in \mathbb{R}, z \in \mathbb{R}\} \end{align}

O alternativamente:

\begin{align} \{(x, y, z) \mid -x + 3y + 2z = 2\} \end{align}

Geométricamente, todas las soluciones están sobre el plano \(-x + 3y + 2z = 2\).

El sistema tiene dos soluciones:

Geométricamente, estas soluciones representan las intersecciones de la línea vertical \(x = 5\) con la parábola \(x = y^2\).

Ejercicio 12: Sistema No Lineal con Círculos ▼
Sistema original:
\begin{align} x^2 + y^2 &= 1 \quad (1)\\ x^2 - 4x + y^2 &= -3 \quad (2) \end{align}
Concepto: Eliminar términos no lineales combinando ecuaciones

Restamos la ecuación (1) de la ecuación (2) para eliminar los términos cuadráticos \(x^2\) y \(y^2\).

\begin{align} (x^2 - 4x + y^2) - (x^2 + y^2) &= -3 - 1\\ x^2 - 4x + y^2 - x^2 - y^2 &= -4\\ -4x &= -4\\ x &= 1 \end{align}
Concepto: Sustituir en una ecuación original

Sustituimos el valor de x en la ecuación (1) para encontrar y.

\begin{align} x^2 + y^2 &= 1\\ (1)^2 + y^2 &= 1\\ 1 + y^2 &= 1\\ y^2 &= 0\\ y &= 0 \end{align}
Concepto: Verificar la solución

Comprobamos que los valores obtenidos satisfacen ambas ecuaciones originales.

Ecuación (1): \(x^2 + y^2 = 1\)

\begin{align} 1^2 + 0^2 = 1 \checkmark \end{align}

Ecuación (2): \(x^2 - 4x + y^2 = -3\)

Ejercicio 13: Sistema No Lineal con Producto ▼
Sistema original:
\begin{align} xy &= 3 \quad (1)\\ x + y &= 4 \quad (2) \end{align}
Concepto: Combinar ecuaciones lineales y no lineales

La ecuación (1) es no lineal debido al producto xy, mientras que la ecuación (2) es lineal. Podemos despejar una variable de la ecuación lineal.

De la ecuación (2), despejamos y:

\begin{align} x + y &= 4\\ y &= 4 - x \end{align}
Concepto: Sustituir en la ecuación no lineal

Sustituimos la expresión para y en la ecuación (1).

\begin{align} xy &= 3\\ x(4 - x) &= 3\\ 4x - x^2 &= 3\\ -x^2 + 4x - 3 &= 0 \end{align}

Multiplicamos por -1 para tener la forma estándar:

\begin{align} x^2 - 4x + 3 &= 0 \end{align}
Concepto: Resolver la ecuación cuadrática

Factorizamos el trinomio para encontrar las raíces de la ecuación cuadrática.

\begin{align} x^2 - 4x + 3 &= 0\\ (x - 3)(x - 1) &= 0 \end{align}

Las soluciones son:

\begin{align} x = 3 \quad \text{o} \quad x = 1 \end{align}
Concepto: Encontrar los valores correspondientes

Para cada valor de x, encontramos el valor correspondiente de y usando la ecuación (2).

Para \(x = 3\):

\begin{align} y &= 4 - x\\ y &= 4 - 3 = 1 \end{align}

Para \(x = 1\):

\begin{align} y &= 4 - x\\ y &= 4 - 1 = 3 \end{align}
Concepto: Verificar las soluciones

Comprobamos que ambos pares de valores satisfacen las ecuaciones originales.

Para \((x, y) = (3, 1)\):

Ecuación (1): \(xy = 3\)

\begin{align} 3 \times 1 = 3 \checkmark \end{align}

Ecuación (2): \(x + y = 4\)

\begin{align} 3 + 1 = 4 \checkmark \end{align}

Para \((x, y) = (1, 3)\):

Ecuación (1): \(xy = 3\)

\begin{align} 1 \times 3 = 3 \checkmark \end{align}

Ecuación (2): \(x + y = 4\)

\begin{align} 1 + 3 = 4 \checkmark \end{align}
Solución:

El sistema tiene dos soluciones:

\begin{align} (x, y) = (3, 1) \quad \text{y} \quad (x, y) = (1, 3) \end{align}

Geométricamente, estas soluciones representan las intersecciones de la hipérbola \(xy = 3\) con la recta \(x + y = 4\).
